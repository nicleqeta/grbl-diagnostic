GRBL BASIC Command Summary

One-page reference for the BASIC runner in this app, grouped by job type.

Line-numbered scripts Case-insensitive commands Numbers + strings

1. Data And Variables

  • LET name = expr: Assign expression result to a variable.
  • READ("$30") INTO name or LET name = READ("$30"): Read a GRBL setting from $$.
  • SETTING("$30") INTO name or LET name = SETTING("$30"): Alias for READ.
  • INPUT name, INPUT name, "Prompt", name = INPUT("Prompt"), LET name = INPUT("Prompt"): Ask user for input.
10 LET max = READ("$30")
20 INPUT speed, "Enter speed"
30 LET pct = INT((speed / max) * 100)

2. Machine I/O Commands

  • SEND expr: Send a command string to GRBL. Expression is converted to text.
  • SEND expr TIMEOUT expr: Wait for terminal acknowledgement up to timeout ms.
  • SEND expr REQUIRE_OK: Fail statement if no ok/error/ALARM acknowledgement arrives.
  • SEND expr TIMEOUT expr REQUIRE_OK: Use both options together.
  • HOLD, RESUME, STATUS, SOFT_RESET: Realtime control commands.
10 SEND "M3 S" & STR(speed) TIMEOUT 1200 REQUIRE_OK
20 STATUS
30 HOLD
40 WAIT 500
50 RESUME

3. Timing And State Waiting

  • WAIT expr: Delay for N ms.
  • WAIT_IDLE or WAIT_IDLE timeoutMs: Wait until machine is idle.
  • WAIT_STATE target or WAIT_STATE target TIMEOUT timeoutMs: Wait for specific state.
10 SEND "G0 X0 Y0" TIMEOUT 1500 REQUIRE_OK
20 WAIT_IDLE 20000
30 WAIT_STATE "idle" TIMEOUT 5000

4. User Messaging And Results

  • PRINT expr: Log a message to terminal.
  • CONFIRM expr: Show confirmation dialog. If canceled, script stops.
  • PAUSE: Pause script until resumed.
  • RESULT name, expr: Save a named result value.
  • REPORT: Print all saved results.
10 PRINT "Starting run"
20 RESULT "target_speed", speed
30 CONFIRM "Run with speed " & STR(speed) & "?"
40 REPORT

5. Program Flow

  • IF condition THEN GOTO line: Conditional jump.
  • GOTO line: Unconditional jump.
  • GOSUB line and RETURN: Subroutine call/return.
  • END: Stop script.
10 LET n = RND(10)
20 INPUT guess, "Guess 1-10"
30 IF guess = n THEN GOTO 80
40 PRINT "Try again"
50 GOTO 20
80 PRINT "Correct"
90 END

6. Expression Functions

  • PI: Circle constant (3.141592653589793).
  • STR(expr): Convert value to string.
  • ABS(expr): Absolute value.
  • INT(expr): Integer truncation.
  • SIN(angleDeg): Sine of an angle in degrees.
  • COS(angleDeg): Cosine of an angle in degrees.
  • TAN(angleDeg): Tangent of an angle in degrees.
  • ASIN(value): Inverse sine, returns degrees.
  • ACOS(value): Inverse cosine, returns degrees.
  • ATAN(value): Inverse tangent, returns degrees.
  • RND(max): Random integer from 1 to max.
  • STATE() or STATE: Current machine state string.
10 LET angle = ATAN(1)
20 LET x = 10 * COS(angle)
30 LET y = 10 * SIN(angle)
40 LET circumference = 2 * PI * 10
50 PRINT "State: " & STATE()

7. Authoring Rules In This App

  • Validation: save is blocked while syntax errors exist.
  • Sharing: saved scripts are loaded by ?script=....
  • Overrides: matching query params override saved variable defaults.
  • Safety: runtime has instruction/time limits and stop controls.
