Supports: LET, READ, SEND, WAIT, WAIT_IDLE, PRINT, INPUT, HOLD, RESUME, STATUS, SOFT_RESET, IF...THEN GOTO, END.
Optional SEND suffixes: TIMEOUT expr waits for an ok/error/ALARM reply up to the given limit, and REQUIRE_OK makes the command fail if no terminal acknowledgement arrives.
Save is blocked when syntax errors are present in the script or variable definitions.
Saved scripts use the title, description, variables, and source in this panel. Share links load via ?script=..., and matching query params override saved variable defaults.
