Tips
Garbled chars on connect? Normal — Arduino bootloader noise. Wait for the Grbl startup line.
Stuck on BUSY? Send $X to clear alarm state.
No ports listed? Install CP2102 or CH340 USB driver.
Chrome / Edge only. Web Serial API is not supported in Firefox or Safari.
