No preset loaded. Shareable test links can prefill diagnostics and safe command sequences.
Builder commands run one at a time in order and stop if the active command returns error: or ALARM:. Use FluidNC-style percent escapes like %3F for special characters, or hex:18 / hex:24 24 0A for raw bytes.

Alarm reference
ALARM:1 Hard limit
ALARM:2 Soft limit
ALARM:3 Reset while in motion
ALARM:4 Probe fail (contact)
ALARM:5 Probe fail (no contact)
ALARM:6 Homing fail (reset)
ALARM:7 Homing fail (door)
ALARM:8 Homing fail (pulloff)
ALARM:9 Homing fail (approach)

Tips
Garbled chars on connect? Normal — Arduino bootloader noise. Wait for the Grbl startup line.
Stuck on BUSY? Send $X to clear alarm state.
No ports listed? Install CP2102 or CH340 USB driver.
Chrome / Edge only. Web Serial API is not supported in Firefox or Safari.
